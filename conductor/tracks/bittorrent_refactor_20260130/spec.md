# Specification: BitTorrent Refactor\n\n## Objective\nImprove media playback performance and reliability by refactoring the BitTorrent service to support background seeding and media chunking.\n\n## Requirements\n- Implement background seeding that persists even when the main UI is not actively viewing media.\n- Add support for media chunking to prioritize the first few megabytes for faster initial playback.\n- Optimize peer-to-peer connection management to reduce CPU overhead.\n- Update UI components to reflect seeding status and buffering progress.
